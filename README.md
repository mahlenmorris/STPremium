# Stochastic Telegraph Premium
Public-facing repo for Stochastic Telegraph's premium module. Hosts docs, public-facing issues, and so on.

![Twixt](images/TwixtHeadline.png)
[Twixt](Twixt.md), a signal generator for VCV Rack, turning a position in a large colorful square into twelve output values.

Includes [Mixt](Twixt.md#mixt), a companion expander module for advanced randomization.
